## Fichiers

- `registre-traitements.md` : registre RGPD interne
- `dpa-template.md` : trame DPA à contractualiser avec chaque client B2B
- `security-annex.md` : annexe sécurité (RPO/RTO/sauvegardes)
- `incident-response.md` : procédure de gestion des incidents
