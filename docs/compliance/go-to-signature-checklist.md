# Checklist "prêt à signature" PME

## Bloc juridique

- [ ] Mentions légales complètes (SIRET, RCS, adresse, directeur publication)
- [ ] CGV/CGU validées
- [ ] DPA prêt à signature
- [ ] Politique confidentialité/cookies à jour

## Bloc sécurité

- [ ] Fiche sécurité client disponible
- [ ] Sauvegardes documentées (fréquence + tests)
- [ ] RPO/RTO définis
- [ ] Process incident documenté

## Bloc commercial

- [ ] Modèle de contrat SaaS B2B
- [ ] SLA opérationnel
- [ ] Modèle de facture conforme
- [ ] RC Pro active

## Bloc technique

- [ ] Isolation multi-tenant validée
- [ ] Exports PDF/Excel validés
- [ ] Alertes email admin validées
- [ ] Audit trail visible dans l'application
