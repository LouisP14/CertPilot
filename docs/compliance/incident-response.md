# Procédure de gestion des incidents

## 1. Détection

- Source: monitoring, logs, alerte client, email de sécurité
- Responsable de triage: À compléter

## 2. Qualification

- Criticité: faible / modérée / majeure / critique
- Périmètre: nombre de clients impactés, données concernées

## 3. Confinement

- Mesures immédiates de réduction d'impact
- Suspension de flux si nécessaire

## 4. Correction

- Correctif technique
- Vérification de non-régression

## 5. Communication

- Information client si impact avéré
- Notification RGPD si violation de données personnelles

## 6. Post-mortem

- Cause racine
- Actions préventives
- Mise à jour documentation sécurité
